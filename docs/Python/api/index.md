# API Reference (Python)

The `nepali-date-library` for Python provides a comprehensive set of tools for working with Bikram Sambat dates.

## Core Components

### [`NepaliDate` Class](./nepali-date-class)
The main class for creating and manipulating Nepali dates. It handles conversion from Gregorian dates, parsing strings, and formatting output.

### [Conversion Functions](./conversion-functions)
Simple utility functions to convert between AD and BS date strings without needing to manage `NepaliDate` instances.

### [Constants](./constants)
Built-in collections of Nepali and English month names, weekday names, and number systems used by the library.
